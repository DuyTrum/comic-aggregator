# 🦕 Dinosaur Icon for Comic Aggregator

## Files Created
- ✅ `dinosaur-icon.svg` - Vector source file (editable)
- ✅ `convert-icon.html` - Browser tool to convert SVG to PNG
- 📦 `appIcon.png` - Main app icon (256x256) - **YOU NEED TO DOWNLOAD THIS**
- 📦 `trayIcon.png` - System tray icon (32x32) - **YOU NEED TO DOWNLOAD THIS**

## How to Use

### Step 1: Download Icons
1. Open `convert-icon.html` in your browser (should already be open)
2. Click "Download appIcon.png" button → Save to this folder
3. Click "Download trayIcon.png" button → Save to this folder

### Step 2: Verify Files
Make sure you have these files in this folder:
```
resources/icons/
├── appIcon.png      (256x256 pixels)
├── trayIcon.png     (32x32 pixels)
└── dinosaur-icon.svg
```

### Step 3: Run the App
The app will automatically use the new icons:
```bash
cd desktop/comic-aggregator
.\comic-aggregator-win_x64.exe
```

## Optional: Create Favicon
To create favicon.ico for better Windows integration:
1. Go to https://www.favicon-generator.org/
2. Upload `appIcon.png`
3. Download the generated `favicon.ico`
4. Copy to `resources/favicon.ico`

## Customize Icon
If you want to change colors or style:
1. Edit `dinosaur-icon.svg` in any vector editor (Inkscape, Figma, etc.)
2. Save the changes
3. Re-open `convert-icon.html` to regenerate PNG files

- 🦕 Cute dinosaur mascot
- 💚 Green color (#66BB6A) - fresh and friendly
- 😊 Smiling face - similar to TruyenQQ penguin style
